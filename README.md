# Men Salon Management System

A Men Salon Management System built using PHP and MySQL. 

## Installation Guide

Follow these steps to run the project on your local environment:

1. **Download and Extract:**
   - Download the project zip file.
   - Extract the file and copy the `msms` folder.

2. **Set up the Server:**
   - Paste the `msms` folder inside your web server's root directory:
     - For XAMPP: `xampp/htdocs`
     - For WAMP: `wamp/www`
     - For LAMP: `/var/www/html`

3. **Database Setup:**
   - Open PHPMyAdmin (`http://localhost/phpmyadmin`).
   - Create a new database named `msmsdb`.
   - Import the `msmsdb.sql` file (found inside the `SQL File` folder in the extracted package).

4. **Run the Application:**
   - Open your web browser and navigate to: `http://localhost/msms`
